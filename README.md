Part of an SSP project for USYD second year physics, where we investigate using double source plane lenses (DSPL) for cosmological constraints. This project builds primarily on the science demonstrated in Collett el al. (2014), extending the results into th w0waCDM cosmology model. We use NUTS MCMC to model constraints on cosmological constants with different lens and sources placements, some DESI results are included (chain1.txt) in order to compare the improvements our model is making on cosmology.
